# 🏦 Banking Transaction Monitoring & Fraud Detection System

## 📌 Overview

A SQL-based project that models a banking system to track transactions and detect fraudulent activities using rule-based logic and automated triggers.

---

## 🎯 Objective

Design a normalized database (3NF) and implement fraud detection mechanisms to identify suspicious transaction patterns in real time.

---

## 🧩 Schema

* Customers
* Accounts
* Transactions
* Fraud Alerts
* Risk Events

**Relationships:**
Customers → Accounts → Transactions → (Fraud Alerts, Risk Events)

---

## ⚙️ Key Features

* High-value transaction detection
* Transaction velocity monitoring
* Location-based anomaly checks
* Trigger-based fraud flagging

---

## 🛠 Tech Stack

* MySQL
* SQL (Joins, Aggregations, Triggers)
* MySQL Workbench (ER Modeling)
